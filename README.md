# Weather Trend Forecasting

## Overview
This project forecasts future weather trends using machine learning models and techniques (preprocessing, cleanign , transformation etc). Make sure to run all cells in order to avoid errors.

## Dataset
GlobalWeatherRepository.csv

## Models Used
- Random Forest
- Gradient Boosting
- Isolation Forest (Anomaly Detection)

## Results
Gradient Boosting performed best with tuned hyperparameters.
